Installation and runing process
please follow these steps

1. Create a directory/folder where you want to run the script
2. Paste final.php and t.csv file in this directory/folder
4. Open terminal in same directory E.g if you are using windows 10, simply double click the adress Bar and TYPE cmd then ENTER
5. Once the command opens, Run this command as: php final.php A E which should give you the out results.



here php is user to run php by command line

final.php is the file which will be run and also consider 0 argument as per php

A is the source argument

B is the target argument
